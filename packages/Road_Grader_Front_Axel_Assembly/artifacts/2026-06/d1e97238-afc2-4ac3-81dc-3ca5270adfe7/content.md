## Description
Isometric CAD view of the road grader front axle assembly. Shows the complete front axle in assembled configuration.

## Key Components Visible

| Component | Color in CAD | Notes |
|---|---|---|
| Axle beam | Yellow | Central structural member |
| Steering knuckles (×2) | Red | Wheel mounting points at each end |
| Tie rod | Blue | Cross-axle steering synchronization link |
| Steering hydraulic cylinder | Silver/grey | Right side of axle beam — primary actuator |
| Wheel hubs (×2) | Yellow (flanged) | Outboard ends, wheel attachment points |
| Kingpin / pivot joints | Yellow | Articulation points at knuckle-to-beam interface |
| Cylinder mounting brackets | Yellow | Transmit steering reaction force into beam |

## Engineering Notes
- Assembly is a rigid front axle configuration (not independent suspension)
- Steering cylinder is single unit, right-side mounted
- Tie rod (blue) runs full axle width — critical for Ackermann synchronization
- CAT part number visible on axle beam casting
